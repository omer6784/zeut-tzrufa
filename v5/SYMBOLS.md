# מסמך הסמלים — פורמט אחיד לקודי p5.js

כל הסמלים של הפרויקט מרוכזים בקובץ אחד: **`v5/symbols-registry.js`**.
מסך התצוגה (`display.html`) קורא מהרישום הזה ובונה את התכשיט בהצטברות —
כל שלב שהמשתמש משלים בממשק מוסיף את הסמל שלו לתצוגה.

---

## מה זה "סמל" בפורמט הזה

כל סמל הוא **פונקציית ציור אחת** שמציירת נקודות (dot-art) בתוך ריבוע
נורמליזציה של `100×100`. הרישום דואג לכל השאר — מיקום על התכשיט, קנה
מידה, צבע, ואנימציה.

### כללי הפונקציה

```js
// כל פונקציית סמל מקבלת:
//   p     — אובייקט p5 (או context תואם) לציור
//   t     — זמן בשניות מאז שהסמל הופיע (לאנימציה עדינה, אופציונלי)
// הפונקציה מציירת בתוך (0,0)–(100,100). הרישום ממקם ומקנה-מידה אותה.
function drawMySymbol(p, t) {
  // לציור נקודה: p.circle(x, y, diameter)
  // הצבע כבר מוגדר ע"י הרישום לפני הקריאה — אל תקבע fill/stroke בעצמך,
  // אלא אם לסמל יש היגיון פנימי של כמה גוונים (ראה "צבעים" בהמשך).
  p.circle(50, 50, 3);
  p.circle(60, 50, 3);
  // ...
}
```

### חוקים
1. **לצייר רק בתוך 0–100.** מה שחורג ייחתך/יעוות את הקומפוזיציה.
2. **לא לגעת ברקע** — אין `background()`. הרקע שייך למסך, לא לסמל.
3. **לא לקבוע צבע קשיח** — צבע הנקודות מגיע מבחוץ (הוא משתנה מפעם לפעם).
   אם חייבים כמה גוונים בתוך סמל, משתמשים ב-`p.__palette` (מערך צבעים
   שהרישום מספק) ולא בקודי צבע קבועים.
4. **נקודות, לא קווים** — השפה היא dot-art. גודל נקודה מומלץ: 2–4 יחידות.
5. **אנימציה אופציונלית** דרך `t` בלבד (למשל ריצוד עדין) — בלי `setInterval`
   ובלי state גלובלי.

## מבנה רשומה ברישום

```js
// symbols-registry.js
export const SYMBOLS = {

  // ── מפתח הסמל (באנגלית, kebab-case) ──────────────────
  'hamsa': {
    name: 'חמסה',              // שם תצוגה בעברית
    stage: 'origin',            // לאיזה שלב בממשק הסמל שייך (ראה טבלה)
    draw(p, t) {
      // ... קוד ה-p5 של הסמל, בתוך 100×100 ...
    },
  },

};
```

### ערכי `stage` (איזה שלב מוסיף את הסמל לתכשיט)

| ערך | שלב בממשק | מיקום על התכשיט |
|---|---|---|
| `origin`   | ארץ מוצא (הגלובוס)   | הגוף המרכזי (seed) |
| `word`     | המילה / הנקודה        | הכתר — האלמנט העליון |
| `stars`    | בחירת הכוכב/סמל       | עמוד הסמלים האמצעי |
| `personal` | המילה האישית          | זוג האלמנטים הצדדיים |
| `roots`    | המסלול (שורשים)       | אלמנט מסלול |
| `name`     | השם                   | הנקודות הסוגרות |

## איך מוסיפים סמל חדש (שלושה צעדים)

1. פותחים את `v5/symbols-registry.js`.
2. מדביקים רשומה חדשה לתוך `SYMBOLS` לפי התבנית למעלה —
   ממירים את קוד ה-p5 שלך לפונקציית `draw(p, t)` שמציירת ב-100×100.
   (אם הקוד המקורי מצייר בקנבס אחר, פשוט מנרמלים קואורדינטות: `x/W*100`.)
3. שומרים. זהו — התצוגה טוענת את הרישום אוטומטית.

## מה הרישום נותן בחינם
- **מיקום והצטברות** — כל סמל מצטרף לתכשיט לפי ה-`stage` שלו.
- **צבע** — צבע הנקודות נקבע במסך (ומשתנה מפעם לפעם), לא בקוד הסמל.
- **אנימציית כניסה** — כל סמל נכנס בעדינות (scale-in), לא בקפיצה.
- **סנכרון חי** — BroadcastChannel/localStorage מהממשק הראשי, כבר מחובר.
